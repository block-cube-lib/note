通常

```rust
fn main() {
    println!("Hello, world!");
}
```

fn main() {
    println!("Hello, world!");
}

行番号付き

```rust,linenos
fn main() {
    println!("Hello, world!");
}
```

1fn main() {

2    println!("Hello, world!");

3}

行ハイライト付き

```rust,hl_lines=2
fn main() {
    println!("Hello, world!");
}
```

fn main() {
    println!("Hello, world!");
}

行番号とハイライト付き

```rust,linenos,hl_lines=2
fn main() {
    println!("Hello, world!");
}
```

1fn main() {

2    println!("Hello, world!");

3}

行番号の開始位置を指定

```rust,linenos,linenostart=10
fn main() {
    println!("Hello, world!");
}
```

10fn main() {

11    println!("Hello, world!");

12}

fn main() {
    println!("Hello, world!");
}
